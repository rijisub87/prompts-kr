## AI 학습의 진짜 가치

ChatGPT를 검색 대용으로만 쓰면 30% 활용입니다. 진짜 가치는 **개인 맞춤 튜터** 역할:

- 막힌 부분을 본인 수준에 맞게 다시 설명
- 끝없이 질문해도 지치지 않음
- 본인 작성 답안을 채점·피드백
- 망각 곡선 따라 복습 일정 짜기

영문 자료에는 잘 정리된 학습 프롬프트가 많지만, **한국 교육 맥락**(수능·내신·자격증·공무원 시험)에 맞게 조정하면 효과가 크게 올라갑니다.

## 시나리오별 패턴

### 1. 모르는 개념을 이해할 때 (Feynman 기법)

```
[개념]을 다음 단계로 설명해줘.

1단계: 초등학생도 이해할 비유로 핵심 1문장
2단계: 중학생 수준 — 왜 중요한지, 어디서 쓰는지 (3~4문장)
3단계: 본격 정의 — 정확한 용어 + 핵심 원리 (구체적)
4단계: 이해 체크 — 본인이 이해했는지 확인할 수 있는 질문 3개

마지막 5번 단계로: 흔한 오해 1~2개와 그 교정.
```

**왜 이게 좋은가**: AI는 두면 첫 줄부터 어려운 정의로 시작합니다. 위 4단계 패턴은 Feynman 학습법을 강제 → 이해 정도 차근차근 올라감.

### 2. 시험·시험 답안 채점

```
다음은 [시험명/과목]에 대한 내 답안이야. 채점하고 피드백해줘.

문제: [문제 전문]

답안:
"""
[본인 답안]
"""

채점 기준 (한국 [내신/수능/공무원시험] 기준):
- 정확성: 사실 오류 있나
- 완성도: 핵심 포인트 다 짚었나
- 표현: 한국식 답안 양식·종결에 맞나
- 점수 (10점 만점)

피드백 형식:
1. 잘된 점 1~2개
2. 부족한 점 + 어떻게 보완할지
3. 모범 답안 (300자 이내)
```

한국 교육의 채점 양식·종결에 맞도록 명시하는 게 핵심입니다.

### 3. 시험 대비 — 예상 문제 생성

```
[교재명/단원] 내용을 바탕으로 시험 예상 문제 5개를 만들어줘.

난이도 분포:
- 쉬움 (개념 확인): 2개
- 보통 (응용): 2개
- 어려움 (통합 사고): 1개

형식: [한국식 객관식 4지선다 / 단답형 / 서술형] 중 [선택]

각 문제 끝에:
- 정답
- 해설 (왜 그 답인지, 다른 보기는 왜 틀린지)
- 출제 의도 — 이 문제로 무엇을 평가하는지
```

### 4. 영어 학습 — 한국인이 어려운 영역

```
다음 영문 [지문/문장]을 한국인 [수능/토익 대비] 관점에서 분석해줘.

원문:
"""
[영문]
"""

분석:
1. 핵심 의미 한국어 번역 (자연스럽게)
2. 한국인이 헷갈리기 쉬운 표현 3개 (왜 헷갈리는지)
3. 비슷한 의미의 다른 표현 2개
4. 실제 사용 예문 2개 (다른 상황)
5. 한국어 직역체와 자연스러운 영어의 차이가 보이는 부분
```

### 5. 학습 일정 짜기 (망각 곡선 활용)

```
다음 학습 목표에 대해 망각 곡선을 고려한 복습 일정을 짜줘.

학습 목표: [목표 — 예: "토익 800점", "정보처리기사 필기"]
가용 기간: [예: 2개월]
주당 학습 가능 시간: [예: 10시간]
현재 수준: [구체적으로]

일정 형식:
- 주차별 학습 + 복습 비율
- 매주 어떤 단원·범위
- 망각 곡선 따라 복습 (1일 후 / 1주일 후 / 1달 후)
- 마지막 1주는 모의고사 + 약점 보완으로
```

## 한국 학습 맥락 — 영문 자료에 없는 차별점

1. **수능 양식**: 객관식 5지선다, 30초 풀이 같은 한국 표준에 맞춰야 함
2. **공무원 시험**: 단답형·약술형이 많음 → 핵심 키워드 위주 답안
3. **내신 vs 수능**: 답안 작성 방식·채점 기준이 다름
4. **자격증**: 기출 변형이 핵심 — AI에 "기출 변형 형식으로" 명시
5. **한국어 글쓰기 평가**: 어휘 다양성·문장 구조 평가 기준이 영어와 다름

## 자주 하는 실수

- **AI에 모든 걸 의존**: 본인이 손으로 풀어보지 않으면 학습 효과 미미
- **답만 받고 끝**: 채점 결과를 받고 "왜 틀렸나"를 한 번 더 묻지 않음
- **너무 일반적인 질문**: "수능 영어 공부법 알려줘" → 일반론. "수능 영어 30문제 중 어법 4문제만 집중하려면" → 구체 답
- **AI 답을 그대로 외움**: AI도 틀릴 수 있음. 핵심 개념은 교과서 한 번 더 확인

## 추천 워크플로 — 1단원 학습

```
1. 교과서 읽기 (30분)
2. AI에게 핵심 개념 Feynman 4단계로 정리 요청 (10분)
3. 본인이 손으로 직접 문제 풀기 (40분)
4. 틀린 문제 AI에 채점·피드백 (10분)
5. AI가 예상 문제 5개 생성 → 다시 풀기 (30분)
6. 1주일 후 복습 알림
```

총 2시간으로 1단원 학습 + 첫 복습까지 완료.

## 마무리

AI는 **24시간 가용한 개인 튜터**입니다. 단, 학원·과외와 다르게 **본인이 능동적으로 질문**해야 가치가 나옵니다. 수동적으로 "정리해줘"만 시키면 정리된 글이 나올 뿐, 학습이 되진 않습니다.

> 핵심: **Feynman 4단계 + 한국 시험 양식 명시 + 채점·피드백 루프 + 망각 곡선 복습 일정**.
