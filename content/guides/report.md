## 직장인 시간을 갉아먹는 3대 보고 작업

다음 3가지가 한국 직장 보고 업무의 90%입니다:

1. **회의록 정리** — 1시간 회의 → 30분 정리 작업
2. **주간/월간 보고서** — 같은 양식 반복 작성
3. **임원 보고용 1페이지 요약** — 가장 어려운 압축 작업

각각 AI에 맡기면 30~50% 시간 절약 가능. 다만 "한국 보고 관례"를 모르는 AI를 어떻게 가르치느냐가 핵심입니다.

## 1. 회의록 자동화

### 워크플로

1. 회의 중 노트북·폰으로 **녹취**(또는 회의 노트 메모)
2. 녹취 텍스트 또는 노트를 AI에 입력
3. 한국 직장 회의록 형식으로 정리 요청
4. 액션 아이템 별도 표로 추출

### 표준 프롬프트

```
다음 회의 내용을 한국 직장의 표준 회의록 형식으로 정리해줘.

회의명: [회의명]
일시: [일시]
참석자: [참석자]

원본 내용:
"""
[녹취록 또는 노트]
"""

출력 형식:
1. 회의 개요 (3줄 이내)
2. 주요 안건별 논의 내용 (각 안건마다 결정사항 명시)
3. 액션 아이템 — 담당자·기한 포함, 표 형태
4. 다음 회의 일정 (언급되었다면)

작성 규칙:
- 격식체, "~함/~임" 끝맺음
- 추측·해석 금지, 회의에서 언급된 내용만 정리
- 액션 아이템에 담당자가 불명확하면 "[담당자 확인 필요]" 표기
- 회의에서 결정되지 않은 사항은 "[결정 보류]"로 명시
```

### 핵심 포인트

- **추측 금지 명시**가 환각 방지의 핵심. 한국 보고에선 근거 없는 추측이 큰 사고로 이어짐
- **"[담당자 확인 필요]" 같은 안전판** — 회의록은 사실 기록이지 추론이 아님
- 녹취가 길면 **Claude나 Gemini 1.5 Pro** (긴 컨텍스트)가 유리

## 2. 주간/월간 보고서

### 가장 흔한 함정

매주 같은 양식의 보고서를 AI에 시키면 **첫 주 결과가 다음 주에 똑같이 반복**되어 결국 사람이 손봐야 합니다. 해결책: **양식과 채울 데이터를 분리**.

```
다음 양식에 맞춰 이번 주 업무 보고서를 작성해줘.

양식:
[양식 그대로]

채울 내용:
- 이번 주 완료: [완료 항목들]
- 진행 중: [진행 항목들]
- 다음 주 계획: [계획]
- 이슈·리스크: [이슈]

작성 규칙:
- 양식의 항목·구조·종결은 그대로 유지
- 추상어("열심히", "최선을 다해") 금지
- 숫자가 있는 것은 숫자로 표기
- 한 항목 50자 이내
```

양식을 매주 재사용하고 채울 내용만 바꿔 입력 → 일관성 + 시간 절약.

## 3. 임원 보고용 1페이지

### 5초 룰

한국 임원은 1페이지를 5초 안에 핵심을 파악해야 합니다. 그래서 **결재자 5초 룰**:

- 800자 이내 (공백 포함)
- 4개 섹션 (배경 / 발견 / 권고 / 다음 단계)
- 각 섹션 200자 이내
- 개조식 "~함/~임" 종결
- 숫자 강조

### 프롬프트

```
다음 정보로 임원 보고용 1페이지 Executive Summary를 작성해줘.

주제: [주제]
배경: [배경 1~2문장]
핵심 메시지: [한 줄로 무엇을 결정 받고 싶은지]

작성 규칙:
- 총 800자 이내 (공백 포함)
- 개조식 종결 ("~함/~임/~예정") 강제
- 4개 섹션: 배경 · 핵심 발견 · 권고안 · 다음 단계
- 각 섹션 200자 이내
- 숫자가 들어갈 자리는 명시 (예: "매출 [X]% 감소")
- BCG·Deloitte 컨설팅 보고 톤
- 마지막에 글자 수 합계 표기

작성 후 다음 5가지 자체 점검:
1. 개조식 종결 일관성
2. 800자 초과 여부
3. 추상어 0회
4. 5초 룰 통과 (한눈에 핵심 보임)
5. 근거 없는 단정 없음
```

"BCG·Deloitte 컨설팅 보고 톤" — 한국에서 LLM에 컨설팅 풍을 시킬 때 가장 잘 통하는 호명입니다.

## 자주 하는 실수

- **녹취 그대로 입력하지 않음**: 본인이 정리해서 넣으면 AI도 정리한 톤으로 답함. 날것을 줘야 정리 가치가 나옴
- **양식과 내용을 매번 같이 보냄**: 양식은 시스템 프롬프트나 Custom GPT로 고정. 내용만 바꿔 입력
- **이중 검수 없음**: AI가 만든 보고서 그대로 임원 보고 → 한 번에 신뢰 무너짐. 항상 본인이 1차 검수 후 제출

## 시간 절약 추정

| 작업 | 수동 | AI 활용 | 절약 |
|---|---|---|---|
| 회의록 정리 | 30분 | 5분 입력 + 10분 검수 | 15분 |
| 주간 보고서 | 1시간 | 15분 | 45분 |
| 임원 1페이지 요약 | 1.5시간 | 30분 | 1시간 |

매주 합치면 **약 2~3시간 절약**. 1년 100시간 이상.

## 마무리

보고는 직장인이 가장 자주 하는 글쓰기지만 가장 보람 없는 글쓰기이기도 합니다. AI에게 한국 보고 관례를 명시적으로 가르치면, 보고에 들이는 시간을 줄이고 더 가치 있는 일에 시간을 쓸 수 있습니다.

> 핵심: **양식·내용 분리 + 추측 금지 명시 + 5초 룰 + 셀프 체크 5단계**.
