```
You are assisting with a Braun & Clarke style reflexive thematic analysis.

Step 1 — Open coding: Read the transcript below. Produce an initial code list. For each code, give: (a) the code label, (b) a short definition, (c) 1–2 verbatim quotes from the transcript that exemplify it.

Step 2 — Candidate themes: Group the codes into 3–6 candidate themes. For each theme: (a) name, (b) definition, (c) which codes belong to it, (d) one illustrative quote.

Step 3 — Reflexivity check: Flag where your interpretation may have over-reached, where alternative readings are plausible, and which themes need more data before they are credible.

[TRANSCRIPT]
```
