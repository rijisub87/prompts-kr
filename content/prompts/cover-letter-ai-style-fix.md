```
아래는 내가 작성한 자기소개서 초안이야. 다음 3가지 관점에서 차례대로 다듬어줘.

1) 문장 구조와 흐름이 자연스럽게 이어지는지 확인하고, 한 문장이 60자를 넘기면 두 문장으로 끊어줘.
2) 중복되는 표현/부사("매우", "정말", "다양한")를 줄이고, 명확하고 가독성 있게 수정해줘.
3) 오탈자와 맞춤법을 올바르게 수정하되, 어떤 부분을 어떻게 수정했는지 표로 정리해줘 (수정 전 / 수정 후 / 이유).

단, 내 경험의 사실관계와 숫자는 절대 바꾸지 말고, AI가 자주 쓰는 표현("끊임없는 노력", "성장의 발판")은 사용 금지.

[자소서 초안]
[DRAFT]

[지원 직무]

```

생성형 AI로 자소서를 쓴 흔적을 지우고 사람이 직접 쓴 글처럼 다듬는 데 쓰는 프롬프트입니다. 문장 흐름·중복 부사·맞춤법을 세 관점으로 나눠 손보고, 마지막엔 수정 전/후/이유를 표로 보여줘서 어디를 왜 고쳤는지 추적할 수 있습니다. "끊임없는 노력", "성장의 발판"처럼 한국 인사담당자가 한눈에 알아채는 상투 표현을 사용 금지로 명시한 게 핵심입니다. 한 번에 다 고치라고 하지 말고 1→2→3 순서로 끊어 실행해야 결과가 안정적입니다. 단, 경험의 사실관계·수치는 바꾸지 말라고 못박았지만 표를 직접 대조해 실제로 보존됐는지 확인하는 게 안전합니다.
