```
Flat vector logo with letter F combined with a flower, modern, Paul Rand style
```

글자 + 모티브 결합형 로고의 기본형. 디자이너 이름 레퍼런스(`Paul Rand`, `Milton Glaser`, `Sagi Haviv`, `Massimo Vignelli`)가 스타일 가이드 역할을 합니다. Midjourney에 붙일 땐 `--no text, gradient --ar 1:1 --v 6`을 추가해 로고 적합도를 높이세요. 단, 결과는 비트맵이므로 실제 사용 전에 벡터(SVG) 트레이싱이 필요합니다.
