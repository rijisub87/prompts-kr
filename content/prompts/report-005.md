```
다음 회의록에서 액션 아이템만 추출해 담당자별로 분배 메시지를 작성해줘.

[회의록 또는 회의 대화 원본 붙여넣기]

출력 형식 (담당자별로 1개 블록):

### [담당자 이름]에게

**TO**: [담당자 이름]
**FROM**: [회의 진행자, 옵션]
**채널**: [슬랙 DM / 이메일 / 노션 등]

본문 (한국어, 정중하지만 친근한 평어체, 5줄 이내):
- 회의 맥락 1줄 (어느 회의 결과인지)
- 본인에게 할당된 액션 아이템 (불릿으로)
- 마감일
- 도움 필요한 경우 회의 진행자/관련자 명시
- 회신 기한 (있다면)

추가:
- 모든 담당자 메시지를 끝낸 뒤, 전체 액션 아이템 1줄 요약 + 미지정 항목 리스트
```

회의록만 정리하고 끝내는 게 한국 직장의 흔한 실패 패턴 — 액션 아이템이 묻혀버립니다. **담당자별 메시지로 자동 변환**하면 회의 진행자가 복붙 한 번으로 분배할 수 있어 실행률이 크게 올라갑니다. 미지정 항목 리스트가 핵심 — 회의에서 빠진 책임 공백을 가시화합니다.
