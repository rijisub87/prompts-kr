```
너는 한국 유튜브 대본 전문 작가야. [VIDEO_LENGTH]분 분량 동영상 대본을 작성해줘.

[영상 정보]
- 주제: [TOPIC]
- 채널 톤: [CHANNEL_TONE] (예: 친근한 일상 / 정보형 / 자극적 후킹)
- 타겟 시청자: [TARGET_VIEWER]
- 영상 형태: [FORMAT] (쇼츠 60초 / 롱폼 8~10분)

[출력 구조]
1. 후킹 (첫 3초): 한국 유튜브 알고리즘 특성상 가장 중요. 질문 / 충격적 사실 / 반전 중 하나 사용
2. 인트로 (10~15초): 채널 소개·이 영상에서 얻을 것
3. 본문 (타임라인 구간별로):
 - 구간별 시간 / 자막용 텍스트 / 카메라 컷·B-roll 제안
 - 한국 시청자가 지루해하지 않도록 30~40초마다 화면 전환 포인트 명시
4. CTA: "구독·좋아요·알림" 자연스럽게 (한국식 멘트, 영문 번역투 X)
5. 썸네일 텍스트 안 (15자 이내) 3개
6. 영상 설명란용 한 문단 + 해시태그 7개

[금지]
- "여러분 안녕하세요" 식 식상한 도입
- 영어 단어 남발 (꼭 필요한 용어 외)
```

주제·채널 톤·타겟 시청자·영상 형태를 입력하면 첫 3초 후킹부터 인트로, 구간별 본문(시간·자막 텍스트·B-roll 제안), CTA, 썸네일 문구, 설명란·해시태그까지 한국 유튜브 문법에 맞춰 한 번에 짜 주는 대본 프롬프트입니다. 영문 대본 프롬프트와 달리 "안녕하세요" 식 식상한 도입과 영어 단어 남발을 금지하고, 30~40초마다 화면 전환 포인트를 명시해 자막 위주로 보는 한국 시청 환경을 반영합니다. 쇼츠 60초든 롱폼 8~10분이든 FORMAT에 맞춰 호흡이 달라집니다. 활용 팁: 후킹은 보통 여러 안으로 나오니 직접 소리 내 읽어 보고 가장 자연스러운 걸 고르고, 채널 톤에는 "정보형이지만 약간 자극적"처럼 결을 구체적으로 적으면 좋습니다. 주의할 점은 모델이 주제 관련 수치나 사례를 그럴듯하게 지어낼 수 있으니, 본문에 들어간 사실 주장은 촬영 전에 따로 확인하세요.
