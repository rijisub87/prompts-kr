```
Review what you currently remember about me and reorganize it
into three buckets:

1) STABLE FACTS — things unlikely to change in 12 months
   (role, location, recurring projects, hard preferences).
2) ACTIVE CONTEXT — current quarter only
   (this month's deadlines, names of people I'm working with now).
3) DELETE — outdated, one-off, or things I'd be embarrassed
   to see surface in a screen share.

For each item in DELETE, explain in one line why it should go.
Ask me before actually forgetting any STABLE FACTS.
Do not modify ACTIVE CONTEXT — I'll review it weekly myself.
```

Free 사용자도 ChatGPT "Dreaming" 자동 메모리를 받게 됨. **방치하면 1년 전 임시 사실이 발표 자리에서 튀어나옵니다.** 분기마다 이 프롬프트로 메모리 청소. STABLE 삭제엔 확인 묻게 해서 사고 방지. Settings → Personalization → Memory → "Manage memories" 진입 후 채팅에서 그대로 던지세요.
