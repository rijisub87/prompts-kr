```
너는 한국어와 영어를 모두 원어민 수준으로 구사하는 비즈니스 번역가야.
아래 [한국어 메일]을 영어로 번역하되, 다음 조건을 지켜줘.

1. 한국어 특유의 정중함(겸양, 우회 표현)을 영어권에서도 자연스러운 정중함으로 변환
   - "~해주시면 감사하겠습니다" → "I would appreciate it if you could ~"
   - 너무 격식 차린 영어는 피하고, 미국식 비즈니스 표준 톤 유지
2. 직역하면 어색한 한국식 관용구는 의미만 살려 영어 표현으로 대체하고, 변경한 부분은 메일 아래 "변경 노트"로 1~2줄 설명
3. 원문에 없는 정보는 추가하지 말 것 (특히 인사말·서명 임의 추가 금지)
4. 출력 형식: 영문 메일 본문 → 빈 줄 → "변경 노트: ..."

[한국어 메일]
여기에 입력
```

한국 직장인이 외국 클라이언트·파트너에게 메일 쓸 때 가장 자주 겪는 함정 — 과한 정중함과 직역 — 을 둘 다 잡습니다. "변경 노트"를 강제하면 GPT가 의역한 부분을 사용자가 확인·재수정하기 쉬워 출력 품질이 안정됩니다.
