```
[서비스 카테고리, 예: 중고거래 / 가계부 앱 / 부동산 매물 플랫폼] 분야의 한국 시장 유사 서비스 3개를 벤치마킹해줘. 가능하면 1개는 글로벌 서비스의 한국 진출 사례를 포함해줘.

다음 항목으로 비교 분석표(마크다운 테이블)를 만들어줘:
| 항목 | 서비스A | 서비스B | 서비스C |
| 핵심 가치 제안 | | | |
| 주요 타겟 (한국 기준 연령·지역) | | | |
| 수익 모델 | | | |
| 차별화 포인트 | | | |
| 약점 / 사용자 불만 빈출 | | | |
| 최근 12개월 주요 업데이트 | | | |

표 아래에:
1. 우리가 진입할 때 비어 있는 시장 틈새(white space) 2개
2. 한국 사용자 특성상 절대 빠뜨리면 안 되는 기능 1개
를 짧게 정리해줘.
```

글로벌 벤치마킹 프롬프트를 그대로 쓰면 한국에 없는 서비스를 비교 대상으로 가져오는 일이 잦습니다. 위처럼 "**한국 시장 유사 서비스**"를 명시하고 "한국 사용자 특성상 빠뜨리면 안 되는 기능"을 요구하면 즉시 현지화된 답이 나옵니다.
