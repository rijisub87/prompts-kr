```
다음 상황으로 거절 메일을 한국어로 작성해줘.

상황: [예: 협업 제안을 받았으나 일정상 어려움 / 견적이 예산 범위를 초과 / 인터뷰 요청 거절]
보내는 사람: [내 직책/회사]
받는 사람과의 관계: [예: 처음 연락받은 외부 업체 / 평소 거래 중인 협력사 / 사내 다른 팀 리더]

원하는 톤:
- 거절 사유를 1가지만 명확히 (변명 나열 금지)
- 상대방 노력에 대한 진심 어린 감사 1줄
- "차후 기회가 된다면" 류 사회적 여지 1줄 (단, 빈말로 들리지 않게 구체적으로)
- "죄송합니다"는 1회만, 과도한 사과 금지

구성:
1. 인사 + 안건 확인 (2줄)
2. 감사 표현 (1줄)
3. 거절 사유 (2~3줄)
4. 향후 여지 또는 대안 제시 (1~2줄)
5. 마무리 인사

분량: 7줄 이내 / 격식체 ("~드립니다", "~겠습니다")
```

한국 직장 거절 메일의 핵심은 **단일 사유 + 절제된 사과 + 사회적 여지** 3박자입니다. AI가 가장 자주 망치는 부분이 "변명 나열"과 "과도한 죄송합니다"라서 위 제약을 명시하는 게 결정적입니다. 사내 메일이라면 "격식체"를 "정중한 평어체"로 바꾸세요.
