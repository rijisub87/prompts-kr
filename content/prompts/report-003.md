```
[주제]에 대한 보고서의 "시사점" 섹션을 작성하려고 해.

분석 결과 요약 (3~5줄):
[데이터, 트렌드, 발견한 패턴]

다음 4가지 관점에서 시사점을 각 2~3개씩 도출해줘:

1. 정책·제도적 시사점 (회사 정책 / 운영 방침 변경 필요성)
2. 전략적 시사점 (사업 우선순위 / 자원 배분)
3. 실행적 시사점 (당장 다음 분기 실행 액션)
4. 리스크 시사점 (이 결과를 무시할 경우 발생할 잠재 손실)

각 시사점:
- 1~2줄로 명확히
- 추상적 진술 금지 ("주목할 필요가 있다" X), 구체적 방향 ("OO 예산을 N% 늘려 ~에 투입할 필요" O)
- 가능하면 정량 근거 1개 첨부

마지막에 "이 시사점들 중 의사결정자에게 가장 먼저 보여줄 1개"를 추천해줘.
```

한국 보고서에서 "시사점"은 가장 자주 두루뭉술해지는 섹션입니다. **4가지 관점 분리 + 추상적 진술 금지** 제약이 결정적. "주목할 필요가 있다" 같은 표현이 GPT 기본 출력에서 자주 나오는데 명시적으로 금지하면 즉시 사라집니다.
