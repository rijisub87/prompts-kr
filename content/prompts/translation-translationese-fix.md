```
다음 한국어 문장은 영어 번역체 흔적이 강합니다. 자연스러운 한국어로 다듬어주세요.

원문 (번역체):
[KOREAN_TEXT]

번역체 신호 (대표적인 것 — 발견 시 모두 제거):
- "~을 통해" — "~으로", "~을 활용해" 등으로
- "~을 가지다" — "~이 있다"
- "~에 있어" — "~에서", "~에"
- "~의 ~의 ~의" 명사 나열 — 동사구로 풀기
- 영어 어순 그대로 (긴 수식어 + 명사)
- 무생물 주어 ("연구는 ~를 보여준다") — "연구에 따르면 ~다"
- 시제 어색함 — 한국어 시제 관행에 맞춤
- "leverage" → "활용하다" 대신 "쓰다/이용하다"
- "best practice" → "모범 사례" 대신 "잘 하는 방법"

출력 3단:
1. 수정본 (자연스러운 한국어)
2. 변경 표 — 원문 / 수정본 / 이유
3. 패턴 일반화 — 이 글 전반에서 반복되는 번역체 1~2가지를 규칙으로 (다음 글에 적용)
```

AI 번역 산출물, 영문 보고서 → 한국어 보고서 변환, 외국계 사내 문서 한국어화에 직격. **패턴 일반화 출력**으로 같은 글의 후속 단락도 일관되게 다듬을 수 있음.
