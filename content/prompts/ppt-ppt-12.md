```
[발표 주제/캠페인 성격, 예: 환경 보호 캠페인] PPT 디자인을 추천해줘.

추천받고 싶은 항목:
1. 색상 팔레트 (메인 컬러 + 보조 컬러 2개, HEX 코드 포함)
2. 추천 폰트 (제목용·본문용 각 1개, 한글 폰트 우선)
3. 슬라이드 레이아웃 구조 (3섹션 또는 2단 구성 등)
4. 시각 강조 요소 (아이콘 스타일·이미지 분위기)
5. 피해야 할 디자인 요소 (지나친 그라데이션·그림자 등)

조건: 청중은 [청중 특성, 예: 일반 시민 / 임원진 / 학생]
```

GPT가 폰트 추천을 영문 위주로 하므로 "한글 폰트 우선"을 명시하지 않으면 Roboto·Open Sans만 추천하고 끝납니다. 한국어 환경에서 자주 등장하는 안전한 폰트(나눔스퀘어, 프리텐다드, KoPub, 산돌고딕네오 등) 중 1개를 받게 됩니다. HEX 코드까지 받으면 파워포인트 "색 채우기 → 더 많은 색 → 사용자 지정"으로 바로 적용 가능.
