```
다음 정보를 임원용 1-Pager 보고서로 압축해줘 (A4 1장, 약 800~1,000자).

원본 정보 (긴 자료 그대로 붙여넣기):
[10페이지 자료 / 회의록 / 분석 보고서 원문]

Co-STAR 프레임워크 적용:
- Context: 임원이 알아야 할 배경 1~2줄
- Objective: 이 보고의 결정 요청 사항 (1줄)
- Style: 한국 기업 임원 보고 양식 (격식체, "~함", "-" 불릿)
- Tone: 객관적·간결·결정 지향
- Audience: 임원 (산업/시장 맥락은 알지만 세부는 모름)
- Response Format:
  1. 핵심 요약 (3줄)
  2. 추진 배경 / 현황 (각 2~3줄)
  3. 의사결정 요청 사항 (예/아니오로 답할 수 있게)
  4. Option A vs B vs C 비교 (마크다운 테이블)
  5. 권고안 + 근거 (2~3줄)
  6. 리스크 / 다음 단계

작성 후, 임원이 받았을 때 결정에 필요한 정보가 빠졌는지 셀프 체크해줘.
```

한국 임원 문화의 핵심은 **"결정만 요청"**입니다. 분석/정보가 아니라 "어떤 선택지 중에 무엇으로 갈지" 명확히 물어야 임원의 시간을 아낍니다. Co-STAR 프레임워크 + 옵션 비교 테이블 + 권고안의 조합이 결정 속도를 크게 올립니다.
