```
너는 네이버 블로그 SEO 전문 작가야. 네이버 C-Rank(주제 전문성)와 D.I.A.+(정보 충실성·독창성·적시성) 알고리즘 기준을 만족하는 글을 작성해줘.

[타겟 키워드]
- 메인 키워드: [MAIN_KEYWORD]
- 서브 키워드: [SUB_KEYWORDS]
- 검색 의도: [SEARCH_INTENT] (정보탐색 / 비교 / 구매)

[작성 단계]
1. 제목 3안 제안 (메인 키워드 앞쪽 배치, 32자 이내, 후킹 포함)
2. 메타 설명 1안 (80자 이내, 메인 키워드 1회 자연 포함)
3. 본문 구성 (h2 소제목 5개, 각 단락 200~300자)
 - 첫 단락에 메인 키워드 자연 포함
 - 서브 키워드는 본문 전체에 골고루 (키워드 스터핑 금지)
 - 실제 경험/사례/숫자 1개 이상 포함 (D.I.A.+ 충족)
 - 마지막 단락은 요약 + 행동 유도
4. 본문 총 분량 1500~1800자
5. 해시태그 10개 (메인 1, 서브 3, 연관 6)

[톤]
- 친근한 반말체("~해요" 톤) 또는 격식체("~합니다") 중 [TONE] 선택
- 광고 같은 단정 표현 ("최고", "유일") 피하기
```

네이버 고유 알고리즘(**C-Rank, D.I.A.+**) 대응은 구글 SEO와 다른 한국 특화 영역. 출력된 본문을 그대로 올리지 말고, 직접 찍은 사진 3~5장을 단락 사이에 끼우면 체류 시간이 올라 노출에 유리합니다.
