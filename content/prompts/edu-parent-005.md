```
이번 주 [학생 이름]([학년]/[반]) 학부모님과 면담이 잡혔어. 면담 준비 자료를 만들어줘.

학생 현재 상황 (3~5줄):
[학습 / 교우 / 태도 / 가정 환경 관련 메모]

학부모 관심사 (사전 파악된 것):
[예: 성적 / 친구 관계 / 진로 / 미디어 사용 습관]

면담 목표:
[예: 학습 동기 회복 / 친구 갈등 해결 협조 요청 / 단순 소통]

출력:
1. 면담 의제 (3~4개, 우선순위 순)
2. 각 의제별:
   - 학교에서 관찰한 객관적 사실 (감정 표현 자제)
   - 학교에서 이미 시도한 조치
   - 부모님께 협조 요청할 내용 (구체적이고 실행 가능)
3. 학부모가 던질 가능성이 높은 질문 5개 + 답변 가이드
4. 면담 마무리 시 다음 단계 제안 (1~2주 후 점검 등)
5. 면담 중 피해야 할 표현 (낙인·평가·비교 톤)
```

한국 교사가 학부모 면담에서 가장 부담스러워하는 부분이 **돌발 질문**입니다. 사전에 예상 질문 5개 + 답변 가이드를 받아두면 즉답력이 크게 올라갑니다. "학교에서 이미 시도한 조치"를 강제 항목으로 둔 게 결정적 — 부모는 학교가 "방관"한다고 느낄 때 가장 격앙되므로 시도 내역을 먼저 제시해야 합니다.
