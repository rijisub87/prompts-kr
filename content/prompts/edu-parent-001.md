```
다음 정보로 우리 아이의 1년 학습 로드맵을 짜줘.

아이 정보:
- 학년: [예: 중학교 1학년]
- 현재 학습 수준: [과목별 간단히, 예: 수학은 1년 선행 / 영어는 학년 평균 / 국어는 약함]
- 일일 학습 가능 시간: [평일 N시간 / 주말 N시간]
- 학사 일정 주요 일자: [중간고사, 기말고사, 수행평가, 방학]

요구사항:
- 수학: 선행 학습 포함 (현재 진도 + N학기 선행)
- 영어: 문법과 독해를 균형 있게
- 국어: 약점 보강 중심
- 그 외 [과목]: [요구사항]

출력 형식:
1. 월별 학습 목표 (마크다운 테이블 — 월/과목/목표/주요 활동)
2. 시험 D-30 / D-14 / D-7 학습 일정 템플릿
3. 방학 중점 보강 계획
4. 매주 부모가 점검할 체크포인트 5개 (사교육 없이도 확인 가능한 항목)

문체: 친근한 평어체, 학부모가 바로 적용 가능한 수준의 구체성
```

사교육 없이 가정 학습을 끌고 가려는 학부모에게 가장 어려운 부분이 **학사 일정 반영**입니다. 중간·기말·수행평가 일자를 직접 입력하면 시험 30/14/7일 전 자동 카운트다운 학습 일정이 나옵니다. 부모 점검 체크포인트 5개가 핵심 — 매주 짧게 아이 진도만 확인해도 사교육 80% 효과.
